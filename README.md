# Cours de Web Avancé - L2 - Université de Lorraine

Ce dépôt contient le cours de Web Avancé L2 donné à l'IDMC Université de Lorraine de 2022 à 2024. Les CM et deux TDs sont des contenus repris et adaptés des versions initiales de Geoffray Bonnin.
À part ces éléments, tout le cours est un contenu original.
